Criação de App de geolocalização com flutter

SGE

Guia Prático: Código & Arquitetura SGE

Este material concentra as etapas necessárias para unificar a captura fotográfica nativa do dispositivo, a extração de dados espaciais via antenas de posicionamento (GPS) e a persistência em banco local.

O foco principal está na manipulação direta dos dados primitivos da imagem na memória para a gravação indestrutível de marca d'água estruturada.

1. Configuração do Arquivo de Dependências

Abra seu arquivo pubspec.yaml localizado na raiz do projeto Flutter e adicione as bibliotecas abaixo na seção de dependências. Elas atuam mapeando os canais de mensagens com o hardware nativo do sistema operacional:

dependencies:
  flutter:
    sdk: flutter
  camera: ^0.10.6              # Controle nativo dos sensores de câmera
  geolocator: ^13.0.1          # Gerenciador de conexão com satélite GPS
  geocoding: ^3.0.0            # Mecanismo de Geocodificação Reversa (Mapeia Coordenadas para Endereço)
  image: ^4.3.0                # Processador e decodificador binário de pixels
  path_provider: ^2.1.3        # Localizador de diretórios seguros do sistema de arquivos
  shared_preferences: ^2.2.3   # Persistência de dados local leve em formato chave-valor

2. Manifesto e Liberação de Recursos Android

Para que o barramento de segurança do Android libere a energia e o uso dos sensores físicos, abra o arquivo android/app/src/main/AndroidManifest.xml. Adicione os metadados de requisição de hardware antes da tag de fechamento do bloco principal:

<uses-permission android:name="android.permission.CAMERA" />
<uses-permission android:name="android.permission.ACCESS_FINE_LOCATION" />
<uses-permission android:name="android.permission.ACCESS_COARSE_LOCATION" />
<uses-permission android:name="android.permission.INTERNET" />

3. Ciclo de Fluxo de Conversação Estrutural

Entender como as camadas operacionais trocam pacotes estruturados é vital para depurar problemas de concorrência ou timeout de hardware:

1. O main.dart pede permissão de hardware ao Android (através do AndroidManifest). 2. O sistema operacional Android autoriza o acesso aos sensores físicos (Câmera/GPS). 3. O framework Flutter intercepta a imagem capturada e extrai os metadados. 4. A biblioteca gráfica 'image' recebe a imagem na memória e substitui as cores dos pixels. 5. O pacote 'path_provider' aponta a pasta segura e grava a imagem permanentemente em disco. 6. A string contendo o endereço exato do disco é injetada e gravada no 'shared_preferences'.

4. Anatomia e Divisão da Lógica Gráfica

A estrutura do arquivo fonte centraliza o controle de estado e telas em quatro blocos lógicos principais:

  • SplashScreen: Carregamento síncrono e identidade visual do software por meio de animação base de progresso linear.
  • HistoricoScreen: GridView dinâmico de duas colunas que lê o armazenamento persistente em busca de arquivos JPEG válidos salvos.
  • VisualizarFotoScreen: Componente adaptável com suporte a escala multiplicativa de toque, focado na leitura detalhada da marca aplicada.
  • CameraSgeScreen: Motor assíncrono que renderiza a visualização da câmera e executa a limpeza de caracteres gráficos não interpretáveis pela fonte bitmap antes de aplicar os métodos fillRect() e drawString().

💡 Alerta Pedagógico ("A Tatuagem de Pixels"): O texto é desenhado diretamente nos bytes binários. Diferente de elementos comuns de interface (Widgets), a fusão dos dados gera um novo arquivo final. Ao transferir este arquivo por qualquer canal, as marcas geográficas permanecem protegidas contra perdas por compressão.

// TELA DE HISTÓRICO (SELEÇÃO MÚLTIPLA)

class HistoricoScreen extends StatefulWidget {
  const HistoricoScreen({super.key});
  @override
  State<HistoricoScreen> createState() => _HistoricoScreenState();
}

class _HistoricoScreenState extends State {
  List<String> _fotosSalvas = [];
  final Set<String> _selecionados = {};

  @override
  void initState() {
    super.initState();
    _carregarHistorico();
  }

  Future<void> _carregarHistorico() async {
    final prefs = await SharedPreferences.getInstance();
    setState(() {
      _fotosSalvas = prefs.getStringList('historico_fotos') ?? [];
      _selecionados.clear();
    });
  }

  void _alternarSelecao(String path) => setState(() => _selecionados.contains(path) ? _selecionados.remove(path) : _selecionados.add(path));

  Future<void> _deletarSelecionados() async {
    final confirmar = await showDialog<bool>(
      context: context,
      builder: (ctx) => AlertDialog(
        title: Text("Deletar ${_selecionados.length} foto(s)?"),
        content: const Text("Esta ação apagará permanentemente os arquivos."),
        actions: [
          TextButton(onPressed: () => Navigator.pop(ctx, false), child: const Text("CANCELAR")),
          TextButton(onPressed: () => Navigator.pop(ctx, true), child: const Text("DELETAR", style: TextStyle(color: Colors.red))),
        ],
      ),
    );

    if (confirmar == true) {
      final prefs = await SharedPreferences.getInstance();
      for (var path in _selecionados) {
        await File(path).delete().catchError((_){});
        _fotosSalvas.remove(path);
      }
      await prefs.setStringList('historico_fotos', _fotosSalvas);
      _carregarHistorico();
    }
  }

  @override
  Widget build(BuildContext context) {
    final ativo = _selecionados.isNotEmpty;
    return Scaffold(
      appBar: AppBar(
        leading: ativo ? IconButton(icon: const Icon(Icons.close), onPressed: () => setState(() => _selecionados.clear())) : const Icon(Icons.map, color: Colors.blueAccent),
        title: Text(ativo ? "${_selecionados.length} selecionadas" : 'SGE - Histórico'),
        actions: ativo ? [
          IconButton(icon: const Icon(Icons.share, color: Colors.blueAccent), onPressed: () => Share.shareXFiles(_selecionados.map((p) => XFile(p)).toList())),
          IconButton(icon: const Icon(Icons.delete, color: Colors.redAccent), onPressed: _deletarSelecionados),
        ] : [],
      ),
      body: _fotosSalvas.isEmpty
          ? const Center(child: Text("Nenhuma foto capturada ainda."))
          : GridView.builder(
        padding: const EdgeInsets.all(8),
        gridDelegate: const SliverGridDelegateWithFixedCrossAxisCount(crossAxisCount: 2, crossAxisSpacing: 8, mainAxisSpacing: 8),
        itemCount: _fotosSalvas.length,
        itemBuilder: (context, idx) {
          final path = _fotosSalvas[idx];
          final isSel = _selecionados.contains(path);
          return GestureDetector(
            onTap: () => ativo ? _alternarSelecao(path) : Navigator.push(context, MaterialPageRoute(builder: (_) => VisualizarFotoScreen(path: path))).then((_) => _carregarHistorico()),
            onLongPress: () => _alternarSelecao(path),
            child: ClipRRect(
              borderRadius: BorderRadius.circular(8),
              child: Stack(
                fit: StackFit.expand,
                children: [
                  Opacity(opacity: isSel ? 0.5 : 1.0, child: Image.file(File(path), fit: BoxFit.cover)),
                  if (isSel) Container(color: Colors.blueAccent.withOpacity(0.3), child: const Icon(Icons.check_circle, color: Colors.white, size: 40)),
                ],
              ),
            ),
          );
        },
      ),
      floatingActionButton: ativo ? null : FloatingActionButton(
        onPressed: () => Navigator.push(context, MaterialPageRoute(builder: (_) => const CameraSgeScreen())).then((_) => _carregarHistorico()),
        child: const Icon(Icons.camera_alt),
      ),
    );
  }
}

// TELA DA CÂMERA E GEOLOCALIZAÇÃO

class CameraSgeScreen extends StatefulWidget {
  const CameraSgeScreen({super.key});
  @override
  State<CameraSgeScreen> createState() => _CameraSgeScreenState();
}

class _CameraSgeScreenState extends State {
  CameraController? _controller;
  bool _estaProcessando = false;
  String _enderecoAtual = "Buscando endereço...", _coordsAtuais = "";

  @override
  void initState() {
    super.initState();
    _inicializarCamera();
    _obterLocalizacao();
  }

  Future<void> _inicializarCamera() async {
    if (_cameras.isEmpty) return;
    _controller = CameraController(_cameras[0], ResolutionPreset.high, enableAudio: false);
    await _controller!.initialize().then((_) { if (mounted) setState(() {}); }).catchError((_){});
  }

  Future<void> _obterLocalizacao() async {
    try {
      if (await Geolocator.checkPermission() == LocationPermission.denied) await Geolocator.requestPermission();
      Position pos = await Geolocator.getCurrentPosition(desiredAccuracy: LocationAccuracy.high, timeLimit: const Duration(seconds: 10));
      setState(() => _coordsAtuais = "Lat: ${pos.latitude.toStringAsFixed(6)} | Long: ${pos.longitude.toStringAsFixed(6)}");
      List<Placemark> p = await placemarkFromCoordinates(pos.latitude, pos.longitude);
      if (p.isNotEmpty) setState(() => _enderecoAtual = "${p.first.street}, ${p.first.subLocality ?? ""} - ${p.first.administrativeArea}");
    } catch (_) {
      setState(() => _enderecoAtual = "Localização indisponível");
    }
  }

  @override
  void dispose() {
    _controller?.dispose();
    super.dispose();
  }

  String _removerAcentos(String text) => text.replaceAll(RegExp(r'[àáâãäåÀÁÂÃÄÅ]'), 'A').replaceAll(RegExp(r'[èéêëÈÉÊË]'), 'E').replaceAll(RegExp(r'[òóôõöÒÓÔÕÖ]'), 'O').replaceAll(RegExp(r'[ùúûüÙÚÛÜ]'), 'U').replaceAll(RegExp(r'[çÇ]'), 'C');

  Future<void> _capturarEProcessar() async {
    if (_controller == null || !_controller!.value.isInitialized) return;
    setState(() => _estaProcessando = true);

    try {
      final XFile foto = await _controller!.takePicture();
      img.Image? imagem = img.decodeImage(await File(foto.path).readAsBytes());

      if (imagem != null) {
        List<String> linhas = [_removerAcentos(_coordsAtuais), _removerAcentos(_enderecoAtual)];
        int rectHeight = (linhas.length * 50) + 35;

        img.fillRect(imagem, x1: 0, y1: imagem.height - rectHeight, x2: imagem.width, y2: imagem.height, color: img.ColorRgba8(0, 0, 0, 200));
        for (int i = 0; i < linhas.length; i++) {
          img.drawString(imagem, linhas[i], font: img.arial24, x: 25, y: (imagem.height - rectHeight + 20) + (i * 50), color: img.ColorRgb8(255, 255, 255));
        }

        final path = "${(await getApplicationDocumentsDirectory()).path}/SGE_${DateTime.now().millisecondsSinceEpoch}.jpg";
        await File(path).writeAsBytes(img.encodeJpg(imagem, quality: 90));

        final prefs = await SharedPreferences.getInstance();
        await prefs.setStringList('historico_fotos', (prefs.getStringList('historico_fotos') ?? [])..insert(0, path));
        if (mounted) Navigator.pop(context);
      }
    } catch (_) {} finally {
      if (mounted) setState(() => _estaProcessando = false);
    }
  }

  @override
  Widget build(BuildContext context) {
    if (_controller == null || !_controller!.value.isInitialized) return const Scaffold(body: Center(child: CircularProgressIndicator()));
    return Scaffold(
      body: Stack(
        children: [
          Positioned.fill(child: CameraPreview(_controller!)),
          Positioned(
            bottom: 120, left: 20, right: 20,
            child: Container(
              padding: const EdgeInsets.all(10),
              decoration: BoxDecoration(color: Colors.black54, borderRadius: BorderRadius.circular(10)),
              child: Column(crossAxisAlignment: CrossAxisAlignment.start, mainAxisSize: MainAxisSize.min, children: [
                Text(_coordsAtuais, style: const TextStyle(color: Colors.white, fontWeight: FontWeight.bold)),
                Text(_enderecoAtual, style: const TextStyle(color: Colors.white)),
              ]),
            ),
          ),
          if (_estaProcessando) Container(color: Colors.black54, child: const Center(child: CircularProgressIndicator())),
          Positioned(bottom: 40, left: 0, right: 0, child: Center(child: FloatingActionButton(backgroundColor: Colors.redAccent, onPressed: _estaProcessando || _coordsAtuais.isEmpty ? null : _capturarEProcessar, child: const Icon(Icons.camera)))),
          Positioned(top: 40, left: 10, child: BackButton(onPressed: () => Navigator.pop(context))),
        ],
      ),
    );
  }
}
